TECNOLOGIAS USADAS.

HTML 5

O HTML5 foi utilizado neste site para criar a estrutura fundamental da página. Ele define elementos como cabeçalhos, parágrafos, links e imagens, proporcionando a base para o conteúdo.

CSS 3

O CSS3 foi empregado para estilizar e dar um visual atraente à página. Controlando elementos como cores, fontes e layout, o CSS3 assegura uma apresentação agradável e coesa.

Java Script

O JavaScript foi implementado para adicionar interatividade à página. Ele permite manipular elementos em tempo real, responder a ações do usuário e facilitar a validação de formulários, proporcionando uma experiência mais dinâmica e envolvente.
